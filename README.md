# Emotion Detection Web Application 🤖💬

## 📌 Project Overview
This project is an AI-based Emotion Detection web application that analyzes customer feedback text and identifies emotions such as **anger, joy, sadness, fear, and disgust**. It is built using Python, Flask, and IBM Watson NLP API (or simulated logic).

---

## 🚀 Features
- Emotion detection from text input
- Identifies multiple emotions:
  - Anger 😡
  - Joy 😊
  - Sadness 😢
  - Fear 😨
  - Disgust 🤢
- Displays dominant emotion
- Web-based user interface using Flask
- Error handling for invalid inputs
- Unit testing support
- Static code analysis (pylint ready)

---

## 🛠️ Technologies Used
- Python 🐍
- Flask 🌐
- HTML/CSS 🎨
